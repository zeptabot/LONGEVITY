# **The Longevity System: Reference Guide**
## TRY IT HERE: https://zeptabot.github.io/LONGEVITY/
What is this?  
This is a responsive, single-page reference tool (LONGEVITY.html) designed to serve as your personal operating system for healthspan extension. It is based on the principles of Medicine 3.0 (from Peter Attia's Outlive) but engineered to adapt to real-world constraints.  
How it Works  
The system is not a static list of rules. It acts as a decision-making engine with two distinct modes of operation:

## **1\. The Idealized State (The "North Star")**

* **When to use:** When you have autonomy over your time and adequate resources.  
* **The Goal:** Optimization. Strive to check every box in this section.  
* **Core Pillars:**  
  * **Movement Engine:** Zone 2 base (3-4h), VO2 Max peak (1x), Heavy Strength (3x), Stability.  
  * **Nutritional Biochemistry:** Protein anchoring (1.6g/kg), Energy balance, Glucose control.  
  * **Restoration:** Sleep opportunity (8h), Circadian sync.  
  * **Emotional/Cognitive:** Social connection, Purpose, Cognitive load.
